Welcome to the NSLS-II Facility API

The NSLS-II Facility API provides programmatic access to beamline, proposal, and user information at NSLS-II. It is intended for staff, beamline scientists, and external collaborators who need to automate data access or integrate with other systems.

  • Browse and test endpoints with interactive documentation
  • Retrieve up-to-date facility and experiment information
  • Integrate seamlessly with your scripts and applications

Get Started

Use the tools below to explore the API or jump right in with our interactive docs.

  • Swagger UI

    An interactive tool to explore, test, and visualize all API endpoints. You can send requests and see live responses directly from your browser.
    • Click View Swagger UI below.
    • Select an endpoint (e.g., /v1/beamlines).
    • Click Try it out, then Execute to see a live response.
    View Swagger UI

  • ReDoc UI

    A comprehensive, user-friendly reference for all endpoints, parameters, and response schemas. Ideal for browsing and understanding the API structure.
    • Click View ReDoc UI below.
    • Browse the sidebar to explore endpoints and models.
    • Click any endpoint to view details, parameters, and response schemas.
    View ReDoc UI
